Configuration

pbjs.setConfig(options)

setConfig supports a number of advanced configuration options:
  • .setConfig(options)
    • debugging
    • bidderTimeout
    • maxRequestsPerOrigin
    • disableAjaxTimeout
    • timeoutBuffer
    • enableSendAllBids
    • useBidCache
    • bidderOrder
    • publisherDomain
    • priceGranularity
    • mediaTypePriceGranularity
    • s2sConfig (server-to-server config)
    • userSync
    • sizeConfig and labels (responsive ads)
    • Generic Configuration
    • Troubleshooting your config
  • .getConfig([string])
